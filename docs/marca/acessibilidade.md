# Acessibilidade

A acessibilidade deve fazer parte do escopo de tudo que criamos no Fab Lab Joinville. Estas diretrizes definem nossos requisitos mínimos.

## Cor

- Nunca dependa exclusivamente da cor para transmitir informação.
- A proporção de contraste entre texto e fundo deve ser de no mínimo `4.5:1`.

## Formulários

- Crie elementos que acompanhem os elementos nativos do navegador sempre que possível.
- Crie estados de foco para campos e botões. Isso melhora a navegação por teclado.
- Rotule todos os campos com clareza. Nunca esconda labels.

## Multimídia

- Sempre forneça texto alternativo para imagens e conteúdo não textual.
- Vídeos devem sempre incluir legendas.

## Estrutura do Documento

- Siga padrões de design estabelecidos sempre que possível.
- Documentos devem estabelecer uma hierarquia clara usando `H1`, `H2`, `H3` etc.
