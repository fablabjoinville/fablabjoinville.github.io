# Acessibilidade

A acessibilidade deve fazer parte do escopo de tudo que criamos. Estas diretrizes definem requisitos mínimos para materiais, interfaces e documentos do Fab Lab Joinville.

## Cor

- Nunca dependa exclusivamente da cor para transmitir informação.
- Garanta contraste mínimo de `4.5:1` entre texto e fundo sempre que possível.
- Combine cor com texto, ícone, padrão ou rótulo quando a informação for crítica.

## Formulários

- Use elementos nativos do navegador sempre que possível.
- Crie estados de foco visíveis para campos e botões.
- Rotule todos os campos com clareza.
- Não esconda labels essenciais.

## Multimídia

- Forneça texto alternativo para imagens e conteúdo não textual.
- Inclua legendas em vídeos sempre que possível.
- Evite depender apenas de áudio ou imagem para explicar uma instrução importante.

## Estrutura do Documento

- Use uma hierarquia clara de títulos (`H1`, `H2`, `H3`).
- Mantenha apenas um `H1` por página.
- Use listas e tabelas quando elas facilitarem a leitura.
- Escreva links com texto descritivo.
