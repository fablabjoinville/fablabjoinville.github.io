# TVC - Thrust Vector Control

![TVC - Thrust Vector Control](/img/projetos/tvc-thrust-vector-control/image-1.jpg)

O projeto de Controle de Vetor de Empuxo (TVC) tem como objetivo desenvolver um sistema capaz de controlar a orientação de um foguete, utilizando motores ou atuadores para desviar o empuxo gerado pelo motor do foguete. Esse controle é essencial para garantir estabilidade, correção de trajetória e pouso seguro. A ideia envolve o uso de sensores, como o MPU6050 para medir aceleração e orientação, e um microcontrolador raspberry pi pico que processa os dados e comanda os atuadores em tempo real. Além disso, o sistema contará com telemetria para enviar informações críticas à estação de solo. Esse projeto é fundamental para adquirir experiência prática em sistemas de controle, telemetria e propulsão, além de abrir portas para futuras aplicações em missões mais complexas.
