# CNC de PCB

![CNC de PCB](/img/projetos/cnc-de-pcb/image-1.jpg)

Utilização de CNC pequena com spindle de alta rotação para a realização de PCBs em de baixa tolerância. O foco atual é a implantação de módulo de nivelamento automático para melhor precisão e dimensionalidade no eixo X.
