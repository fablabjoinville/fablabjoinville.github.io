# Code like a girl

![Code like a girl](/img/programas/code-like-a-girl/image-1.png)

O Fab Lab Joinville em parceria com o Catainas UDESC organizou um laboratório de programação em Python voltado para meninas jovens e adultas. O evento ocorreu ao longo de quatro sábados, a partir de 19 de Maio, proporcionando uma imersão em programação com foco no desenvolvimento de aplicações web.

Este não foi um curso destinado a iniciantes na programação, sendo recomendado que as participantes já tivessem algum contato prévio com a linguagem. No entanto, para aquelas sem experiência, foram anunciados planos para realizar novos laboratórios nos meses seguintes, oferecendo oportunidades para quem estava começando a se aventurar na área.

Durante o evento, as participantes tiveram a oportunidade de aprofundar seus conhecimentos em Python e desenvolver habilidades práticas por meio de projetos direcionados. A ênfase foi dada à aplicação prática dos conceitos aprendidos, visando capacitar as participantes para o desenvolvimento de suas próprias aplicações web.

**Durante o Evento:**

- **Foco da Imersão:** Proporcionou uma imersão em programação com foco no desenvolvimento de aplicações web.

- **Oportunidades Futuras:** Planos foram anunciados para realizar novos laboratórios nos meses seguintes, oferecendo oportunidades para quem estava começando a se aventurar na área.

- **Ênfase na Aplicação Prática:** Foi dada ênfase à aplicação prática dos conceitos aprendidos; Capacitar as participantes para o desenvolvimento de suas próprias aplicações web.
